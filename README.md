Parse data from various files and then create a excel file as a report

Install commands:

```
virtualenv -p python3 venv
# On Windows
source/Script/activate
# On Linux
source/bin/activate

pip install pyqt5 pywebview 
```


Package commands:

```
pip install fbs pyinstaller
fbs startproject
fbs run
fbs freeze
```
